# Blog Posts

## Quick Reference

**Purpose:** Thought leadership content that drives engagement and action
**Length:** 800-1500 words
**Formality:** Medium (professional but accessible)
**Audience:** Industry professionals, decision-makers, practitioners
**Best frameworks:** Inverted Pyramid, SCQA
**Key requirement:** Bold opening hook, clear call-to-action

## Writing Style Guidelines

### Opening Hook

- First paragraph MUST be bold (`**text**`)
- Sets urgency and frames the challenge/opportunity
- 2-3 sentences maximum
- Should make reader want to continue

### Section Headings

- Use German section titles for German blogs (as shown above)
- Each heading introduces a clear theme
- Logical flow: Why → What → How → Risks → Benefits → Action

### Within-Section Formatting

- **Bold subheadings** for each point within sections
- Numbered lists (1., 2., 3., 4.) for transformation/change items
- Bullet lists for risks and benefits
- Keep explanatory text to 1-3 sentences per point

### Tone and Voice

- Professional but accessible
- Urgency without fear-mongering
- Evidence-based claims (reference source data)
- Active voice (80%+ target)
- Direct, punchy sentences (15-20 words average)

## Key Characteristics

- **Engaging hook:** Bold opening grabs attention immediately
- **Logical flow:** Problem → Solution → Action progression
- **Evidence-based:** Claims backed by data from source insight
- **Action-oriented:** Clear call-to-action closing
- **Scannable:** Bold subheadings, lists, short paragraphs

## Visual Elements

**Visual density target:** ~1 visual element per 2-3 paragraphs

**Primary visual elements:**

- Bold text for emphasis and subheadings
- Numbered lists for sequential items
- Bullet lists for parallel items (risks, benefits)
- Bold metrics within bullet points

**Formatting rules:**

- Bold opening paragraph (entire first paragraph)
- Bold each subheading within sections
- Bold metrics/numbers in benefit lists
- Commas at end of list items, period on final item

## Quality Standards

- 800-1500 words total
- Flesch 50-60 readability
- 80%+ active voice
- 3-5 items per numbered/bullet list
- Every claim traceable to source insight

## TIPS-Derived Content Guidelines

When copywriting blogs generated from TIPS insights:

**Preserve:**

- All quantitative claims from source (percentages, metrics, timeframes)
- Evidence trail (paraphrased, not academic citations)
- Core message alignment with insight's Trend/Implications/Possibilities/Solutions

**Enhance:**

- Sentence flow and readability
- Call-to-action language strength
- Professional tone consistency
- Bold formatting for scanability

**Do NOT:**

- Invent facts not in source insight
- Add unsubstantiated claims
- Change quantitative data
- Remove evidence-based statements
