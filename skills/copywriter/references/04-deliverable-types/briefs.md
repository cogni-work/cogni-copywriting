# Professional Briefs

## Quick Reference
**Purpose:** Concise decision support for executives
**Length:** 1-3 pages
**Formality:** Medium-high
**Audience:** Executives, decision-makers
**Best frameworks:** BLUF, Pyramid, SCQA
**Reading time:** 2-5 minutes maximum

## Standard Structure

### With BLUF (Decision Brief)
```markdown
# [Decision Required]: [Topic]

BLUF: Recommend [decision] by [date] to [achieve outcome].

Context: [2-3 sentences of essential background]

Analysis:
- Option A: [pros/cons]
- Option B: [pros/cons]

Recommendation: [Clear recommendation with rationale]

Next Steps: [What happens after approval]
```

### With Pyramid (Strategic Brief)
```markdown
# [Topic] Strategic Brief

## Answer
[Main conclusion/recommendation]

## Three Key Findings
1. Finding 1
2. Finding 2
3. Finding 3

## Supporting Analysis
[Evidence for each finding]

## Recommendations
[Actionable next steps]
```

## Key Characteristics

- **Extreme conciseness:** Every word earns its place
- **Executive-optimized:** Scannable in 2-3 minutes
- **Decision-focused:** Clear recommendation, not just analysis
- **Evidence-based:** Data supports conclusions

## Visual Elements

**Visual density target:** 1-3 visual elements (tables for comparisons, bullets for lists)

**See also:** [visual-elements.md](../03-formatting-standards/visual-elements.md) - General visual formatting

## Quality Standards

- 1-3 pages maximum
- Flesch 50-60 (standard difficulty)
- 3-5 sentences per paragraph
- Visual elements: tables for comparisons, bullets for options
