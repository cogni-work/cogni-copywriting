# Business Letters

## Quick Reference
**Purpose:** Formal external correspondence
**Length:** 1 page typically
**Formality:** Very high
**Audience:** External stakeholders, formal contexts
**Best frameworks:** Direct (good news) or Indirect (bad news) approach
**Key requirement:** Professional formatting, formal tone

## Standard Structure

```markdown
[Your Letterhead]

[Date]

[Recipient Name]
[Title]
[Company]
[Address]

Dear [Mr./Ms. Last Name]:

[Opening: Purpose statement]

[Body: Details and context, 2-3 paragraphs]

[Closing: Next steps or courteous close]

Sincerely,

[Your Signature]
[Your Typed Name]
[Your Title]
```

## Framework Choice

**Direct Approach** (Good news, routine requests):
- Main point first paragraph
- Details follow
- Like BLUF framework

**Indirect Approach** (Bad news, persuasion needed):
- Context first
- Build to main point
- Like SCQA framework

## Key Characteristics

- **Formal language:** No contractions, formal salutations
- **Professional formatting:** Block or modified block style
- **Complete address blocks:** Sender and recipient
- **Dated:** Always include date
